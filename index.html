Roster Database

Import a clean ISS export with these columns: LRN, Complete Name, Gender, Grade Level, Section, Subject. The app uses LRN as the student identity and treats Grade Level + Section + Subject as the class roster.

Attendance is saved by Date + Grade Level + Section + Subject + LRN. This prevents Mathematics attendance from overwriting Science or English attendance for the same section.

Import from ISS CSV

The import merges new students and roster memberships. Existing attendance records are kept.

Backup and Restore

Use this before moving to another browser, computer, or tablet. The backup includes all students, rosters, settings, and all saved attendance records currently stored on this device.

Teacher Quick Guide

  1. Install once while online. Open the GitHub Pages link, then add the app to the tablet Home Screen.
  2. Import the ISS roster CSV. Use the format LRN, Complete Name, Gender, Grade Level, Section, Subject.
  3. Record attendance by class. Select the date, grade level, section, and subject before marking learners.
  4. Saved records are locked. Unlock editing only when you intentionally need to correct a saved record.
  5. Back up weekly. Tap Backup Now and save the JSON file in Files, Google Drive, or a computer folder.
  6. Do not clear website data without a backup. Clearing Safari or Chrome site data can delete the local tablet database.
  7. Restore before using a new device. Use the latest JSON backup to bring back students, rosters, and attendance records.
